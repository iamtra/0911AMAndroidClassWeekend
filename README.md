# 0911AMAndroidClassWeekend

# Jetpack Compose Masterclass

Welcome to the Jetpack Compose Learning Hub, a project designed to transition students from XML-based 
layouts to modern, declarative UI development on Android. This repository serves as a comprehensive guide,
containing practical examples, UI components, and best practices for building robust Android applications.

# About the Project
This project is curated specifically for students and beginners. It breaks down complex UI patterns 
into digestible, modular code. Whether you are learning state management or complex animations, 
this repository provides the "blueprint" for modern Android engineering.

# Key Learning Modules:
    
- Composables & Modifiers: Understanding the building blocks of UI.
- State Management: Mastering remember, mutableStateOf, and ViewModel integration.
- Theming: Deep dive into Material Design 3 (M3) and Dark Mode support.
- Lists & Grids: Efficiently handling data with LazyColumn and LazyVerticalGrid.

# Requirement
- Language: Kotlin 2.0+
- Gradle JDK: JetBrains Runtime 21
- minSdk = 29

# Getting Started 
1. Clone the repository:
    git clone  https://github.com/iamtra/0911AMAndroidClassWeekend.git
2. Open in Android Studio:
   Wait for the Gradle sync to finish.
3. Run on Device:
   Connect a physical device or an emulator (API 30+) and hit Run.

# Recommendations for Students

To get the most out of this project, I recommend:

1. Don't just copy-paste: Try to rewrite the UI components from scratch while referring to the source code. 
2. Experiment with Modifiers: Change padding, background colors, and alignment to see how the UI reacts in real-time with Compose Preview. 
3. State Observation: Use the Layout Inspector in Android Studio to see how recomposition works.
